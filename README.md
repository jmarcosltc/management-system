# GRWN Management System

This is a simple management system built for small business.

Still in development.


## Authors

- [@jmarcosltc](https://www.github.com/jmarcosltc)


## Installing

Install with npm or yarn

```bash
  npm install
  cd management-system
```

```bash
  yarn install
  cd management-system
```
